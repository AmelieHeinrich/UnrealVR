Contenu du Projet
Le projet se divise en trois mini jeu distincts, chacun explorant un genre spécifique :

1. Beat Saber Lite (Rythme)
Une version simplifiée du célèbre jeu de rythme.

Mécanique : Des blocs défilent vers le joueur.

Objectif : Détruire les blocs à l'aide de sabres (ou collisions simples).

État actuel : Génération basique de cubes, score basique.

2. Urban Climber (Escalade)
Un prototype axé sur la physique des mains et la gestion de la verticalité.

Mécanique : Système d'accroche sur des briques.

Objectif : Atteindre le sommet d'un mur simple sans lâcher prise.

État actuel : Mur 'simple', prises fonctionnelles et parcours basique.

3. Micro Kart (Course)
Un mini-jeu de conduite avec une physique "arcade".

Mécanique : Contrôle d'un véhicule sur un plan horizontal.

Objectif : Rouler.

État actuel : left shoot pour monter sur le kart et descendre (n'importe ou), main bouge gauche ou droite pour tourner.
